# Gratitude.com — Illustration & Art Direction Guide

## Visual Identity

### Core Aesthetic
Dark, luminous, glow-driven aesthetic. Pure black canvases with pink and orange
energy radiating from within. The design language is warm technology: where the
precision of a platform meets the emotion of gratitude.

### What We Look Like
- Pure black backgrounds with pink/orange glow orbs and energy
- Gradient accent lines that pulse with warmth
- Depth through layered glows: pink, coral, and orange light on dark void
- Grid patterns at low opacity suggesting structure and scale
- High contrast white text against dark, with careful opacity management
- The 6-component hover system bringing interactive warmth to all elements

### What We Never Look Like
- Light or white backgrounds (except rare contrast CTAs)
- Navy or teal backgrounds (that's 561 Media, not Gratitude)
- Robot hands, humanoid AI, or generic "tech" imagery
- Stock photo feel or cluttered designs
- Cartoon or clip-art style illustrations
- Neon/cyberpunk aesthetics (our glow is warm, not cold)
- Muted earth tones or pastel palettes
- Charity-style imagery (sad faces, hands reaching out)

## Design Elements

### Background Treatments
1. **Pure black** (#000000) — Default. The void from which glow emerges.
2. **Near black** (#0a0a0a) — Slightly elevated sections.
3. **Hero gradient** — Layered radial pink/coral/orange glows on black.
4. **Grid overlay** — 60px grid at 3% white opacity for structural texture.
5. **Glow compositions** — Large blurred pink/orange orbs creating depth and warmth.

### Accent Elements
- **Signature gradient lines:** Pink→coral→orange, 1-4px height, used as section accents
- **Glow orbs:** Large blurred circles of pink (rgba(254,49,132,0.15-0.30)) floating in compositions
- **Orange secondary glows:** Smaller orbs (rgba(236,114,17,0.10-0.15)) for depth
- **Grid patterns:** Subtle 60px grid at 3% opacity as background texture
- **Bottom accent bars:** Gradient lines that scale in on hover

### Card & Container Style
- Background: Gradient from #1a1a1a to #0d0d0d (top to bottom)
- Border: 1px solid rgba(255,255,255,0.08)
- Border-radius: 1rem (16px)
- Hover: Full 6-component system (glow orb, text brightness, icon scale, icon glow, bottom accent, border glow)
- Shadow: 0 20px 60px rgba(0,0,0,0.4), 0 0 80px rgba(254,49,132,0.1)

### Button & CTA Style
- Primary: 3-stop gradient (pink→coral→orange), pill shape (border-radius: 9999px)
- Text: White, Inter SemiBold (600)
- Shadow: 0 10px 40px rgba(254,49,132,0.3)
- Hover: translateY(-2px), shadow intensifies
- Secondary: Ghost button with 1px white/20% border

### Icon Style
- Thin line weight (1.5px stroke)
- Color: Pink (#FE3184)
- Container: Pink/10% background, pink/20% border, rounded-xl
- Hover: Container border strengthens, glow shadow appears, icon scales 110%
- Consistent sizing within a set (w-7 h-7 default)

## Composition Principles

### Visual Hierarchy
1. One hero element per composition (the thing you see first)
2. Supporting elements clearly subordinate in size and weight
3. Dark negative space is a design tool — it lets the glow breathe
4. The eye should travel: hero → supporting info → CTA

### Layout Patterns
- **Centered:** Hero text centered, supporting elements below. Best for social posts.
- **Left-weighted:** Large element left, supporting content right. Best for LinkedIn.
- **Grid:** 2-3 column card layouts with consistent spacing.
- **Asymmetric:** Dynamic off-center layouts for campaign key art.

### Color Usage Rules
- Black is the canvas. It covers 60-70% of most compositions.
- Pink is the primary accent. Used for labels, icon glows, primary energy.
- Orange is the secondary accent. Used for gradient endpoints and warmth.
- Gradient is the signature. Pink→coral→orange appears on CTAs, accent lines, text highlights.
- White text with opacity scale creates the entire text hierarchy.
- Never use colors outside the defined palette.

## Platform-Specific Notes

### Social Media Graphics
- Instagram: Bold, high-contrast, minimal text on black. Pink glow energy.
- LinkedIn: Professional, data-forward, black background with cleaner layout.
- Stories: Vertical compositions. Account for UI overlay zones.

### PDF Deliverables
- Cover pages: Dramatic black with glow effects and gradient accents.
- Interior pages: Can use dark or light backgrounds depending on content density.
- Consistent header/footer treatment throughout.
- Page numbers in Inter, bottom center or bottom right.

### Web Mockups
- Follow the brand guide CSS patterns exactly.
- Cards, buttons, sections must match the gratitude.com design system.
- Include the full 6-component hover system on all interactive elements.
- Use real brand gradients, shadows, and radii from visual-system.json.

### Email Graphics
- Max 600px wide. Optimize file size (under 100-150KB).
- High contrast for email client rendering.
- Simple compositions that work without images loaded.
- Dark background with pink/orange accents.

### How to Use It

1. Go to **agents.gratitude.com** and enter the password
2. Pick an agent from the sidebar (or start with the **Orchestrator** if you're not sure)
3. Type your request and the response streams back in real-time
4. Your conversations are saved — switch to the **History** tab to pick up where you left off
5. Use **"+ New Chat"** to start a fresh conversation with the same or a different agent

---

### Workflow Chains

The agents are designed to chain together. Some common pipelines:

- **Social content with graphics:** Content Atomizer → Social Creative
- **Landing page with design:** Direct Response Copy → Web Mockup
- **Lead magnet PDF:** Lead Magnet → Deliverable Design
- **Sponsor kit:** Direct Response Copy → Deliverable Design
- **Full campaign:** Positioning Angles → Direct Response Copy → Content Atomizer → Social Creative

The Orchestrator knows all of these and will recommend the right chain for your goal.

### Knowledgebase

There's a **Knowledgebase** page (link in the sidebar footer) that stores learnings from conversations. It's pre-loaded with 23 entries covering our positioning, voice rules, terminology, visual system specs, and workflow documentation. As you have conversations, the system automatically extracts new learnings after 4+ messages and adds them to the knowledgebase. Future conversations pull from this, so the agents get smarter over time as the team uses them.

### Limitations to Know

- **Design agents are in "Web Mode"** — You'll see an orange banner on the 5 design agents. They can provide detailed design specs, creative direction, copy, color values, typography, and layout descriptions, but they cannot render actual PNG or PDF files in the browser. For rendered output, those agents need to be run locally through Claude Code. Use them here for specs and direction, then bring that to the local tool for production assets.
- **Streaming responses** — Responses stream in real-time. If you navigate away mid-stream, the partial response is still saved.
- **Session length** — Your login session lasts 8 hours before you'll need to re-enter the password.
- **Model** — All agents run on Claude Sonnet 4, which is fast and capable. The brand context and knowledgebase are injected into every conversation automatically.
- **Shared password** — Everyone uses the same password. There are no individual accounts. Conversation history is shared across the team, so you can see and continue each other's conversations.
